About WebRats

We are a secure search engine, nothing is logged. The only thing we monitor is if the website it up, we ping it every 30 seconds. Nothing else is saved

Why use it?

  • WebRats may not offer you as personalised results as Google, but it doesn't generate a profile about you.
  • WebRats doesn't care about what you search for, never shares anything with a third party, and it can't be used to compromise you.
  • WebRats is free software, the code is 100% open and you can help to make it better..

Technical details - How does it work?

WebRats is a metasearch engine, inspired by the seeks project. WEBRATS.XYZ IS CONNECTED TO OVER 3000 PROXY SERVERS. Everytime you make a request we shoot your request out to one of the 3000 proxies in the network, fetch your results all in under 1 second (pretty cool right). It provides basic privacy by mixing your queries with searches on other platforms without storing search data. Queries are made using a POST request on every browser (except Chromium-based browsers*). Therefore they show up in neither our logs, nor your url history. In the case of Chromium-based browser users there is an exception: searx uses the search bar to perform GET requests. WebRats can be added to your browser's search bar; moreover, it can be set as the default search engine.

How to set as the default search engine?

WebRats supports OpenSearch. For more information on changing your default search engine, see your browser's documentation:

  • Firefox
  • Microsoft Edge
  • Chromium-based browsers only add websites that the user navigates to without a path.
